# Task Management System

ระบบจัดการรายการงานแบบหน้าเว็บ  
พัฒนาด้วย HTML, CSS และ JavaScript  
ใช้กระบวนการ DevOps ร่วมกับ GitLab และ GitLab CI/CD

## Features
- เพิ่มรายการงาน
- แสดงรายการงาน
- แก้ไขรายการงาน
- ลบรายการงาน

## DevOps Tools
- GitLab Repository
- GitLab CI/CD
- GitLab Pages
- Unit Testing (JavaScript)

## Branch Strategy
- main : สำหรับ production
- dev  : สำหรับพัฒนา
